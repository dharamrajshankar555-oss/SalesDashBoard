### **Project Overview**
This project involves the creation of a dynamic and interactive **Sales Dashboard** using **Microsoft Excel**. The dashboard provides a comprehensive 360-degree view of sales performance, profitability, and product trends across various international markets. 

It is designed to help stakeholders visualize key metrics (KPIs) and make data-driven decisions by filtering data dynamically through Slicers.

---

### **‚ú® Key Features**
* **Dynamic Filtering:** Integrated **Slicers** allow users to filter the entire report by **Segment, Country, Product, Year,** and **Month** instantly.
* **KPI Cards:** High-level metrics displayed at the top for quick insights:
  * **Total Sales**
  * **Total Profit**
  * **Total Quantity Sold**
  * **Total Discounts**
* **Visual Analysis:** * **Sales by Country:** Geographic breakdown of revenue.
  * **Profit by Product:** Identifying high-margin products.
  * **Segment Analysis:** Performance comparison between Government, Enterprise, Midmarket, etc.
  * **Monthly Trend:** Line/Bar charts tracking performance over time (2013-2014).
* **Custom UI/UX:** A modern, dark-themed interface for better visual contrast and professional presentation.

### **üìÇ Dataset Description**
The raw data contains sales transactions with the following key attributes:

| Column Name | Description |
| :--- | :--- |
| **Segment** | The market segment (e.g., Government, Enterprise) |
| **Country** | The location of the sale (e.g., Canada, Germany, USA) |
| **Product** | The name of the item sold (e.g., Carretera, Velo) |
| **Units Sold** | Quantity of items sold |
| **Gross Sales** | Revenue before deductions |
| **Profit** | Net earnings from the transaction |
| **Date** | Transaction date (spanning 2013-2014) |

3. The charts and KPI cards will update automatically to reflect the selected criteria.
